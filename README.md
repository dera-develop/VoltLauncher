# VOLT LAUNCHER
exeファイルを実行するアプリです．  
アプリ名とexeファイルの絶対パスを事前に登録することで，簡単にアプリを呼び出すことが出来ます．  
「自作した便利ツール等のアプリを一か所で自由に起動したい」ということで作成しました．

## 実装機能
基本機能として，  
・実行アプリの登録/更新  
・登録アプリの実行  
・登録アプリの削除  
・登録アプリ一覧の表示  
の4種類の機能が実装されています．  
  
### 実行アプリの登録/更新
  アプリを登録します．  
  ```
  vlncr add [app_name] [app_path]
  # app_name: アプリの識別子を入力します
  # app_path: アプリの絶対パスを入力します
  ```
  登録した内容は，スクリプトファイルと同層の ./paths.json に {"識別名": "絶対パス"} のように保存されます．  
  このファイルを直接編集する事でも登録が出来ます．  
  ファイルが存在しない場合は自動的に生成されます．  
  
### 登録アプリの実行
  登録されているアプリを実行します．  
  ```
  vlncr run [app_name] [app_args]
  # app_name: 登録したアプリの識別子を入力します
  # app_args: 実行アプリに渡したい引数を入力します．
  ```
  app_name 以降の引数は全て実行時にアプリケーションに渡される引数となります． 
  ```
  vlncr run exampleApp arg1 arg2...  
  ```
  となる場合， 
  ```
  exampleApp arg1 arg2...
  ```
  として実行されます．  
  
### 登録アプリの削除 
  登録リストからアプリを削除します．  
  ```
  vlncr del [app_name]
  # app_name: 削除するアプリの識別子を入力します
  ```
  
### 登録アプリ一覧の表示
  登録リストを表示します．  
  ```
  vlncr list
  ```
